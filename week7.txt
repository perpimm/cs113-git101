การใช้งานโโปรแกรม git
1. git version เป็นการตรวจสอบว่าโปรแกรมสามารถใช้งานได้ปกติหรือไม่ 
2. git init เป็นการสร้าง git repository ใหม่บนเครื่องของเรา
3. git status เป็นการตรวจสอบสถานะของไฟล์เราจะใช้ git status เมื่อเราเพิ่มไฟล์ใหม่ ลบไฟล์ไฟล์ หรือมีการแก้ไขไฟล์เดิม
4. git add เป็นการเพิ่มไฟล์เข้าไป
5. git config user.name , email เป็นการตั้งชื่อผู้ใช้และระบุอีเมลผู้ใช้
6. git commit -m เป็นการเก็บ versionจองไฟล์แล้วมีข้อความระบุว่าเป็นไฟล์เกี่ยวกับอะไร
7. git log --all --decorte --onelone --graph เป็นการ checkpoint ข้อมูลทั้งหมด
8. git remote add เป็นการเชื่อมเครื่องเรากับเว็บไซต์github
9. git push -u origin master เป็นการนำไฟล์งานของเราไปวางบนเว็บไซต์ของ github
10. git checkout คือการนำไฟล์กลับมาทำงานใหม่
